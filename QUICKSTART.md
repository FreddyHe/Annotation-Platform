# 快速使用指南

## 🎯 使用流程

### 第一步：启动服务

```bash
cd /root/autodl-fs/web_biaozhupingtai
./run.sh
```

服务启动后，访问 `http://localhost:6006` 或通过AutoDL端口映射访问公网地址。

### 第二步：创建项目

1. 在左侧边栏的"项目管理"区域
2. 输入项目名称（例如：`animal_detection`）
3. 点击"创建/切换项目"

### 第三步：定义类别

1. 在左侧边栏的"类别定义"区域
2. 设置类别数量（例如：3个类别）
3. 为每个类别输入：
   - **类别名称**：例如 `dog`, `cat`, `bird`
   - **类别定义/Prompt**：详细描述该类别的特征，用于VLM验证
     ```
     例如：狗是一种四足哺乳动物，通常有毛发，是人类常见的宠物...
     ```
4. 点击"保存类别定义"

### 第四步：上传数据

在"数据上传"标签页：

**方式1：上传ZIP压缩包**
- 选择"文件夹(ZIP)"
- 上传包含图片的ZIP文件
- 系统会自动解压并扫描图片

**方式2：直接选择文件夹**
- 选择"直接选择文件夹"
- 输入图片文件夹路径（例如：`/root/autodl-fs/my_images`）
- 点击"扫描文件夹"

### 第五步：批量处理

1. 切换到"批量处理"标签页
2. 确认：
   - ✅ 已上传图片
   - ✅ 已定义类别
3. 选择处理选项：
   - **启用VLM清洗**：推荐开启，使用大模型进行二次验证
   - **批处理大小**：建议10-20张
4. 点击"🚀 开始批量处理"
5. 等待处理完成（会显示进度条）

### 第六步：查看结果

1. 切换到"结果查看"标签页
2. 在下拉框中选择要查看的图片
3. 查看：
   - 带标注框的可视化图片
   - 统计信息（总检测数、高分保留、VLM验证等）
   - 检测详情表格

### 第七步：导出标注

1. 切换到"标注编辑"标签页
2. 选择导出格式：
   - **COCO JSON**：标准COCO格式，适合大多数训练框架
   - **YOLO**：YOLO格式，包含labels文件夹和classes.txt
3. 点击"导出标注文件"
4. 下载或查看保存路径

## 📁 输出文件位置

所有结果保存在 `/root/autodl-fs/web_biaozhupingtai/data/` 目录下：

```
data/
├── uploads/
│   └── <项目名>/
│       └── extracted/          # 解压后的图片
├── processed/
│   └── <项目名>/               # 处理后的数据
└── annotations/
    └── <项目名>/
        ├── annotations.json    # COCO格式标注
        └── labels/             # YOLO格式标注（如果导出YOLO）
```

## ⚙️ 配置调整

如需修改配置，编辑 `config.py` 文件：

```python
# 修改VLM服务地址
VLM_BASE_URL = "http://your-vlm-server:port/v1"

# 调整检测阈值
GD_HIGH_SCORE_THRESH = 0.65  # 高分阈值（>=此值直接保留）
GD_LOW_SCORE_THRESH = 0.3   # 低分阈值（<=此值直接丢弃）

# 修改端口
SERVER_PORT = 6006
```

## 🔍 常见问题

### Q1: 处理速度慢怎么办？

**A**: 
- 减少批处理大小
- 关闭VLM清洗（但会降低准确率）
- 确保GPU可用

### Q2: VLM服务连接失败？

**A**: 
- 检查 `config.py` 中的 `VLM_BASE_URL` 是否正确
- 确认VLM服务是否正常运行
- 检查网络连接

### Q3: 检测结果不准确？

**A**: 
- 优化类别定义/Prompt，提供更详细的描述
- 调整阈值参数（`GD_HIGH_SCORE_THRESH` 和 `GD_LOW_SCORE_THRESH`）
- 确保类别名称与Grounding DINO能识别的词汇匹配

### Q4: 如何批量处理大量图片？

**A**: 
- 建议分批处理，每批100-200张
- 处理完成后及时导出结果
- 清理session_state以释放内存

### Q5: 标注编辑器如何使用？

**A**: 
- 当前版本主要支持查看和导出
- 交互式编辑功能正在开发中
- 如需手动修正，可以：
  1. 导出COCO格式
  2. 使用其他标注工具（如LabelMe）进行编辑
  3. 重新导入

## 💡 使用技巧

1. **类别定义要详细**：好的类别定义能显著提升VLM验证准确率
2. **合理设置阈值**：根据实际检测效果调整高分和低分阈值
3. **分批处理**：对于大量图片，建议分批处理并保存中间结果
4. **及时导出**：处理完成后及时导出，避免session过期导致数据丢失
